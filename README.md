Welcome!

This repository contains the supplementary materials for the paper:
“Blind Binaural Speech-Intelligibility Estimation from Reverberant Speech for Edge Acoustic Sensing” (Alfian et al., 2026)

It provides the following resources:
1) BRIR datasets, consisting of 2800 BRIRs from 115 rooms, captured from 9 databses/research groups.
2) Python source code.

All materials in this repository are open-source.
If you use any of these resources, please cite the paper (Alfian et al., 2026).

Thank you.
